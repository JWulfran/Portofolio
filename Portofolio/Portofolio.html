Analyse RH

Project Portofolio

+

Préférence pour le travail flexible

Année : 2025

Analyse RH · Machine Learning

Ce projet explore les préférences des employés pour le travail flexible à partir d’un jeu de données de 9 000 individus. En s’appuyant sur des algorithmes de classification (KNN, arbres de décision, réseaux de neurones), il identifie les facteurs clés comme le temps de déplacement, le burnout ou la satisfaction au travail.

Informations

Le modèle C5.0 a offert les meilleurs résultats. Cette analyse permet aux décideurs RH d’adapter leurs politiques internes, en ciblant les profils les plus enclins à opter pour des modalités flexibles.

Technologies
  • Python (Pandas, Scikit-learn, Keras)
  • Visualisation : Matplotlib, Seaborn
  • Jupyter Notebook

Housing Predicton

Project Portofolio

+

Prédiction du prix des maisons

Année : 2025

Régression · Machine Learning

Ce projet vise à prédire les prix des maisons à partir de données réelles en utilisant des algorithmes de machine learning. Les étapes incluent le nettoyage des données, l’analyse exploratoire, la sélection de variables pertinentes et l’entraînement de modèles (régression linéaire, random forest, etc.).

Informations
  • Problématique : prédire le prix des logements à partir de caractéristiques structurées (surface, nombre de pièces, année de construction, etc.).
  • Type de tâche : régression supervisée.
  • Modèles évalués : Régression linéaire, Ridge/Lasso, Arbre de décision, Random Forest.
  • Métriques d’évaluation : RMSE, MAE, R².
  • Validation & optimisation : validation croisée et réglage d’hyperparamètres via GridSearchCV.
  • Meilleur modèle : Random Forest Regressor (bon compromis biais/variance sur l’échantillon de test).
Technologies
  • Python
  • Pandas, NumPy
  • Scikit-learn
  • Matplotlib, Seaborn

Calculatrice simple

Programmation

+

Calculatrice DIY

Année : 2025

Application web · Programmation

Description

Petite application web permettant d’effectuer des opérations arithmétiques usuelles avec une interface minimaliste et réactive. Idéale pour démontrer la gestion d’événements, la manipulation du DOM et la séparation structure/style/comportement.

Fonctionnalités clés
  • Opérations : addition, soustraction, multiplication, division.
  • Touches : chiffres, point décimal, Clear, retour arrière.
  • Gestion des erreurs (division par zéro, saisies invalides).
  • Clavier & clic : support des deux modes d’entrée.
Informations
  • Type : Application monopage (SPA légère)
  • Architecture : HTML sémantique + CSS modulaire + JS vanilla
  • Tests rapides : scénarios d’enchaînement d’opérations et d’annulation
  • Accessibilité : focus visible, aria-labels, boutons claviers
Technologies
  • HTML, CSS
  • Javascript
